# Recipe Web App

A Flask-based web application that allows users to explore recipes using TheMealDB API.

## Features
- Search recipes by category, area, ingredient, or name
- Random recipe generator
- Dynamic pages with Flask & Jinja templates

## Tech Stack
- Python (Flask)
- HTML / CSS / Bootstrap
- REST API integration

## Setup

1. Clone the repository
2. Install dependencies:
pip install -r requirements.txt

3. Create a .env file and add your API key:
API_KEY=your_key

4. Run the app:
python main.py
